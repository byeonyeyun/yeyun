## Projects

### 1. 흡연 시각화 프로젝트
- 폴더: `흡연 시각화 프젝`
- 내용: 건강검진 데이터를 바탕으로 흡연 여부에 따른 주요 지표 차이를 시각화하고 해석한 프로젝트
- 핵심 작업: EDA, 분포 비교, 상관관계 분석, 스토리텔링 중심 시각화

### 2. 흡연 예측 모델
- 폴더: `흡연 예측 모델`
- 내용: 건강 관련 수치를 활용해 흡연 여부를 예측한 이진 분류 프로젝트
- 핵심 작업: 데이터 탐색 및 전처리, 모델링, 확률 예측, 분류 성능 검토

### 3. 임신 성공 여부 예측
- 폴더: `임신 성공 여부 예측`
- 내용: 임신 성공 여부를 예측하기 위한 머신러닝/오토ML 실험 프로젝트
- 핵심 작업: 데이터 탐색 및 전처리, 피처조정, 모델링, 앙상블 가중치 실험

### 4. 심리 성향 예측
