# Data Analysis Portfolio

취업용 포트폴리오로 정리한 데이터 분석 및 머신러닝 프로젝트 모음입니다.  
주피터 노트북 기반으로 EDA, 시각화, 분류 모델링, 앙상블 실험을 진행한 결과물을 담고 있습니다.

## Projects

### 1. 흡연 시각화 프로젝트
- 폴더: `흡연 시각화 프젝`
- 내용: 건강검진 데이터를 바탕으로 흡연 여부에 따른 주요 지표 차이를 시각화하고 해석한 프로젝트
- 핵심 작업: EDA, 분포 비교, 상관관계 분석, 스토리텔링 중심 시각화
- 주요 산출물:
  - `흡연 시각화 1조.ipynb`
  - `흡연 시각화 1조 PPT.pdf`

### 2. 흡연 예측 모델
- 폴더: `흡연 예측 모델`
- 내용: 건강 관련 수치를 활용해 흡연 여부를 예측한 이진 분류 프로젝트
- 핵심 작업: 데이터 전처리, 분류 모델링, 확률 예측, 분류 성능 검토
- 확인된 키워드: ROC-AUC, Classification Report, 제출용 예측 생성
- 주요 산출물:
  - `흡연 예측모델 11조.ipynb`
  - `흡연 예측모델 11조 PPT.pdf`

### 3. 임신 성공 여부 예측
- 폴더: `임신 성공 여부 예측`
- 내용: 임신 성공 여부를 예측하기 위한 머신러닝/오토ML 실험 프로젝트
- 핵심 작업: AutoGluon 기반 학습, 피처 조정, 앙상블 가중치 실험, 제출 파일 생성
- 확인된 키워드: AutoGluon, LightGBM, XGBoost, CatBoost, Weighted Ensemble, ROC-AUC
- 주요 산출물:
  - `bestbestmodel.ipynb`
  - `오토글로운_피쳐5_나이_drop_2hr(0.7411+0.74215)-.ipynb`
  - `feature_5_2hr_drop_autogluon_submission.csv`

### 4. 심리 성향 예측
- 폴더: `심리 성향 예측`
- 내용: 심리 성향 분류/예측을 주제로 수행한 프로젝트
- 핵심 작업: 앙상블 기반 예측 실험 및 제출 결과 비교
- 주요 산출물:
  - `ensemble_3.ipynb`
  - `final_masterpiece_ensemble_2.csv`
  - `심리 성향 예측 8조 PPT.pdf`
